# IIVTrade — Dental Supplies Trading & Distribution

⚠️ This template was originally created by Mohammed Sabry
credit is required: github.com/Mohammedsabry13


A professional one-page corporate website for **IIVTrade**, a dental supplies trading and distribution company established in 2014, supplying clinics and markets with high-quality dental instruments and tools.

## 🛠️ Tech Stack

- **HTML5 & CSS3** — semantic one-page structure and custom styling
- **Bootstrap 5.3** — responsive grid and navbar
- **Swiper.js 11** — hero slider and partners carousel
- **Font Awesome 6.4** — icon set
- **Lucide Icons** — lightweight SVG icon set
- **DM Serif Display + Manrope** — Google Fonts elegant typography
- **Vanilla JavaScript** — comparison slider, stats counter, Swiper init

## ✨ Features

- **Swiper.js hero slider** — 2 fullscreen slides with background images, overlay, badge, animated title, feature checklist items and CTA buttons
- **Sticky navbar** — with social icons (Facebook, Twitter, LinkedIn, Google+) and smooth scroll navigation
- **About section** — image with floating spinning tooth badge, feature checklist and "Get To Know" label
- **Services section** — 3 service cards with images, icons and descriptions
- **Before & After comparison slider** — interactive drag handle to compare before/after project images
- **Dedicated Health section** — 2-column benefits checklist with brand story (since 2014)
- **Partners/Brands carousel** — Swiper.js logo slider for dental brand partners
- **Animated stats counter** — key company numbers with count-up animation on scroll
- **Contact section** — inquiry form with name, email, subject and message fields
- **Footer** — with logo, quick links, social icons and copyright
- Fully responsive across all screen sizes
